# noCap

A multilingual slang and tone analyzer built with a local LLM (LM Studio).

## Features
- Detects slang across English, Hinglish, and mixed languages
- Intent-aware slang detection
- Avoids false positives
- Works with local models (Mistral)

## Tech Stack
- Frontend: Vanilla JS
- Backend: Node.js + Express
- LLM: LM Studio
